# 🤖 Arduino Projects Collection

Welcome to my Arduino Projects repository! 🚀
This repository contains several beginner-to-intermediate level projects developed using Arduino.

Each project focuses on learning electronics, sensors, automation, and basic programming concepts.

---

## 📂 Projects Included

### ⚠️ Gas Detector

* Detects gas presence using a gas sensor.
* Useful for safety and environmental monitoring.

### 🌗 Light Intensity Control

* Adjusts system behavior based on light levels.
* Uses a light sensor (LDR).
* Great for automation projects.

### 🔆 LED Board

* Controls multiple LEDs.
* Useful for practicing digital outputs.

### 🔴 LED Project

* Basic LED control project.
* Demonstrates blinking and output logic.

### 🚦 Traffic Light

* Simulates a real traffic light system.
* Uses timed LED sequences.

### 🚪 Ultrasonic Door Sensor

* Detects object distance using an ultrasonic sensor.
* Can be used for automatic doors or security systems.

## 📌 How to Use

1. Open the desired `.ino` file in **Arduino IDE**.
2. Connect your Arduino board.
3. Select the correct board and port.
4. Upload the code.
5. Assemble the hardware according to the project requirements.
